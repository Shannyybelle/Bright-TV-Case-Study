# Bright-TV-Case-Study

## Project Overview
This project is an end-to-end data analytics case study focused on BrightTV, a fictional streaming platform. The objective is to analyze user profiles and viewing behaviour to uncover insights that can help increase subscriptions and improve customer engagement.

The project includes data cleaning, transformation, exploratory data analysis (EDA), and dashboard development using SQL, Databricks, and Power BI.

---

## Business Problem

BrightTV wants to grow its subscription base by understanding:

- Who their users are
- What content users watch
- When users watch content
- Which audience segments are most engaged
- Opportunities to improve user retention and increase subscriptions

---

## Objectives

- Clean and prepare raw datasets
- Identify data quality issues
- Analyze user demographics
- Analyze viewing behaviour
- Discover consumption patterns
- Build an interactive dashboard
- Provide business recommendations

---

## Dataset

The project uses two datasets:

### User Profiles
Contains demographic information including:

- User ID
- Name
- Email
- Gender
- Age
- Province
- Race
- Social Media Handle

### Viewership
Contains streaming activity including:

- User ID
- Channel
- Program
- Start Time
- End Time
- Viewing Duration
- Viewing Date
